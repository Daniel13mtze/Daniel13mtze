### 📈 Proyectos Destacados

#### 🎮 [HCI Hardware-Software Interface](https://github.com/Daniel13mtze/unity-arduino-serial-interface)
*Sistema de Instrumentación Virtual e Interfaz Interactiva en Tiempo Real*
- **¿Qué es?:** Un sistema desarrollado durante mi proyecto de residencias profesionales que integra hardware electrónico con un entorno virtual 3D enfocado en la rehabilitación física.
- **Logros Técnicos:** 
  - Programación de la lógica y control de físicas en **C#** dentro de **Unity Engine**.
  - Programación del firmware en **C++ (Arduino)** para procesar lecturas dinámicas de acelerómetros y giroscopios.
  - Sincronización de datos mediante comunicación serial por USB en tiempo real para el control fluido del software.

#### 🏥 [Hospital Management System](https://github.com/Daniel13mtze/hospital-management-system)
*Aplicación de Escritorio y Modelado de Bases de Datos*
- **¿Qué es?:** Una herramienta interactiva de gestión hospitalaria desarrollada en **C# (.NET Windows Forms)** para centralizar flujos operativos clave de una institución médica.
- **Logros Técnicos:** 
  - Arquitectura modular dividida en 5 pilares operativos mediante eventos síncronos.
  - Backend de persistencia estructurado en **SQL Server** para el control de citas, flujos financieros, índices de pacientes (NP) e historiales clínicos.
